# Bot_v_0-1 — MVP Telegram бота учёта продуктов

Минимальная версия бота для ведения списка продуктов в холодильнике.

Цель версии:
создать рабочий прототип, позволяющий вручную добавлять продукты и получать уведомления об истечении срока годности.

---

# Основные функции

1. регистрация пользователя
2. добавление продукта вручную
3. категории продуктов
4. хранение продуктов в холодильнике
5. уведомление о сроке годности
6. просмотр содержимого холодильника
7. удаление продуктов

---

# Команды Telegram

/start  
регистрация пользователя

/add  
добавить продукт

/fridge  
показать продукты

/remove  
удалить продукт

/expiring  
продукты с истекающим сроком

/help  
помощь

---

# Пользовательский сценарий

Добавление продукта:

1 пользователь нажимает /add  
2 выбирает категорию  
3 вводит название продукта  
4 вводит дату истечения  
5 продукт сохраняется

# Уведомления

Каждый день выполняется задача:

SELECT fridge_items
WHERE expiration_date <= today + 3

Бот отправляет сообщение пользователю.
