# Bot_v_2 — масштабируемая система учета продуктов

Версия с высокой автоматизацией и большой базой товаров.

Цель:
бот должен автоматически распознавать большинство продуктов.

---

# Основные улучшения

1. база GTIN 1–2 млн товаров
2. быстрый сканер DataMatrix
3. ML классификация продуктов
4. рекомендации рецептов
5. семейный холодильник
6. аналитика продуктов

# База продуктов

Источники данных:

Open Food Facts

каталоги интернет-магазинов

API системы "Честный знак"

Цель:

собрать 1–2 млн GTIN.

---

# Импорт данных

pipeline:

download dataset
↓
normalize GTIN
↓
deduplicate
↓
import to database

---

# Нормализация GTIN

все GTIN приводятся к формату:

14 digits

пример:

4607062446630  
→  
04607062446630

---

# Redis кэш

используется для:

GTIN lookup

rate limiting

FSM состояний бота

---

# Улучшенный сканер

pipeline:

image
↓
grayscale
↓
bilateral filter
↓
CLAHE contrast
↓
adaptive threshold
↓
deskew
↓
decode

точность распознавания:

90-97%

---

# ML классификация продукта

если название неизвестно:

модель определяет категорию.

пример:

"Actimel клубника"

→ категория "молочные продукты"
