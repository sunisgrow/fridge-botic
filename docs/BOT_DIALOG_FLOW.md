# Telegram Bot Dialogue Flow

Conversation map for the Fridge Bot.

---

## Main Menu

### Entry Point
User sends: `/start`

Bot response:
```
👋 Добро пожаловать в Fridge Bot!

Я помогу отслеживать продукты в вашем холодильнике.

Выберите действие:
```

Keyboard:
```
┌─────────────────┬─────────────────┐
│   ➕ Добавить   │   📷 Сканировать │
├─────────────────┼─────────────────┤
│   🧊 Холодильник│   ⚠️ Истекает    │
├─────────────────┼─────────────────┤
│   🍳 Рецепты    │   🛒 Покупки     │
├─────────────────┴─────────────────┤
│            ⚙️ Настройки           │
└───────────────────────────────────┘
```

## Flow: Add Product (Manual)

### Step 1: Category Selection
User presses: `➕ Добавить`

Bot response:
```
Выберите категорию продукта:
```

Keyboard:
```
┌──────────┬──────────┐
│ 🥛 Молочные│  🥩 Мясо  │
├──────────┼──────────┤
│ 🥬 Овощи  │  🍎 Фрукты│
├──────────┼──────────┤
│ 🥤 Напитки│  🧊 Замор.│
├──────────┼──────────┤
│ 🫗 Соусы  │  🍞 Хлеб  │
├──────────┴──────────┤
│      📦 Другое       │
└─────────────────────┘
```

### Step 2: Product Name
User selects: `🥛 Молочные`

Bot response:
```
Введите название продукта:

Например: Молоко 3.2%, Йогурт клубничный
```

User input: `Молоко 3.2%`

### Step 3: Brand (Optional)
Bot response:
```
Укажите бренд (или пропустите):
```

Keyboard:
```
┌──────────────┬───────────────┐
│  Простоквашино │  Домик в деревне│
├──────────────┼───────────────┤
│    Valio     │    Danone     │
├──────────────┴───────────────┤
│          ⏭️ Пропустить        │
└─────────────────────────────┘
```

User selects: `Простоквашино`

### Step 4: Expiration Date
Bot response:
```
Введите срок годности:

Формат: ДД.ММ.ГГГГ
Например: 20.03.2026
```

User input: `15.04.2026`

### Step 5: Quantity
Bot response:
```
Укажите количество:
```

Keyboard:
```
┌─────┬─────┬─────┬─────┬─────┐
│  1  │  2  │  3  │  4  │  5  │
└─────┴─────┴─────┴─────┴─────┘
```

User selects: `2`

### Step 6: Confirmation
Bot response:
```
✅ Продукт добавлен:

📦 Молоко 3.2% (Простоквашино)
📂 Категория: Молочные
📅 Срок годности: 15.04.2026
📊 Количество: 2 шт.
```

Keyboard:
```
┌─────────────────┬─────────────────┐
│   ➕ Ещё один   │   🧊 В холодильник│
└─────────────────┴─────────────────┘
```

## Flow: Scan Product (DataMatrix)

### Step 1: Request Photo
User presses: `📷 Сканировать`

Bot response:
```
📷 Отправьте фото упаковки с DataMatrix кодом.

Убедитесь, что код хорошо виден и освещён.
```

### Step 2: Processing
User sends photo

Bot response:
```
⏳ Обрабатываю изображение...
```

### Step 3a: Product Found
Bot response:
```
✅ Продукт распознан:

📦 Молоко 3.2% (Простоквашино)
📅 Срок годности: 15.04.2026
🔢 GTIN: 04607062446630

Подтвердить добавление?
```

Keyboard:
```
┌─────────────────┬─────────────────┐
│   ✅ Да, добавить│   ✏️ Изменить   │
└─────────────────┴─────────────────┘
```

### Step 3b: Product Not Found
Bot response:
```
❓ Продукт не найден в базе.

GTIN: 04607062446630

Введите название продукта:
```

User input: `Новый продукт`

Bot response:
```
Выберите категорию:
[Категории...]

✅ Продукт добавлен в библиотеку!
```

### Step 3c: Scan Failed
Bot response:
```
❌ Не удалось распознать код.

Возможные причины:
• Плохое освещение
• Код размыт
• Код повреждён

Попробуйте снова или добавьте вручную.
```

Keyboard:
```
┌─────────────────┬─────────────────┐
│   📷 Повторить  │   ➕ Вручную    │
└─────────────────┴─────────────────┘
```

## Error States

### Network Error
```
❌ Ошибка соединения

Проверьте интернет и попробуйте снова.
```

### Invalid Input
```
❌ Неверный формат

Введите дату в формате ДД.ММ.ГГГГ
Пример: 15.04.2026
```

### Product Not Found
```
❌ Продукт не найден

Попробуйте изменить поиск или добавьте вручную.
```

## State Machine

```
IDLE
  ↓ /add
SELECTING_CATEGORY
  ↓ category selected
ENTERING_PRODUCT_NAME
  ↓ name entered
ENTERING_BRAND
  ↓ brand selected/skipped
ENTERING_EXPIRATION
  ↓ date entered
ENTERING_QUANTITY
  ↓ quantity entered
CONFIRMING
  ↓ confirmed
IDLE
```
